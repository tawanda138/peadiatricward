🛠️ Cloudflare Proxy — Free 5-min Setup

You need this because your browser cannot talk directly to DHIS2 (CORS restriction). This free proxy routes the request for you — no IT department needed.

  1. Go to cloudflare.com → sign up free (just an email)
  2. Click Workers & Pages → Create Application → Create Worker
  3. Delete all the default code in the editor
  4. Paste in the code from the dhis2-proxy-worker.js file (download below)
  5. Click Save and Deploy
  6. Copy the worker URL shown (e.g. https://dhis2-proxy.yourname.workers.dev)
  7. Paste that URL into the Cloudflare Proxy URL field above
  8. Click Connect — done ✅
Free tier: 100,000 requests/day — more than enough for monthly reporting.
